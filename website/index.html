Description

BudgetNow is a simple budgeting tool that allows you to manually track income and expenses in custom budgeting categories. You can create as many transactions and budgets as you'd like, but your budgets will only reflect transactions for the current month.

With some great features:

  • Create unlimited custom budget categories
  • Create unlimited transactions, and categorize each to a budget
  • Review your current months spending
  • And its 100% free!
